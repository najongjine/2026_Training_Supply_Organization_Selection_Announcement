# 특성화산업 직종 정리 1

## 디지털 문화콘텐츠 분야

### 1. 영상 콘텐츠 제작

- 직무 개요: 영화, 드라마, 광고, 웹콘텐츠 등 영상물의 기획, 촬영, 편집, 후반작업을 수행하는 직종
- 주요 훈련 내용
  - 시나리오 작성
  - 촬영 실무
  - 영상 편집
  - 색보정 및 후반작업
  - 사운드 디자인
  - 특수효과(VFX)

### 2. 게임 콘텐츠 제작

- 직무 개요: 모바일, PC, 콘솔, 메타버스 기반 게임의 기획과 개발을 수행하는 직종
- 주요 훈련 내용
  - 게임 기획 및 레벨디자인
  - Unity, Unreal Engine 활용
  - C#, C++, Blueprint 기반 개발
  - 3D 그래픽 제작
  - 게임 UI/UX
  - QA 및 최적화

### 3. 스마트 문화콘텐츠 제작

- 직무 개요: 모바일·웹 기반 문화콘텐츠 앱, XR(VR·AR·MR), 메타버스용 콘텐츠를 기획·설계·제작하는 직종
- 주요 훈련 내용
  - 문화콘텐츠 기획
  - UI/UX 설계
  - 프로토타이핑
  - Flutter, React Native 기반 앱 개발
  - Unity XR, Vuforia 기반 XR 콘텐츠 제작
  - 360도 영상 및 메타버스 공간 설계

## 반도체 첨단 패키징 분야

### 4. 공정장비 운영

- 직무 개요: 시스템반도체 패키징 공정에 활용되는 장비의 운영 및 공정 적용을 다루는 직종
- 예시 공정
  - 백그라인딩
  - 라미네이션
  - 웨이퍼 소잉(Wafer Sawing)
- 주요 훈련 내용
  - 패키징 공정 장비 이해
  - 장비 운용 실습
  - 장비 유지관리
  - 반도체 공정 적용 실무
